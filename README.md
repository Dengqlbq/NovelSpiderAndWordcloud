# NovelSpiderAndWordcloud
Scrapy练习项目

###目标：爬取起点小说X类型小说前X页的所有小说并将所有简介做成词云

###power by：
1. Python 3.6
2. Scrapy 1.4
3. pymysql 
4. wordcloud
5. macOS 10.12.6

####博客地址：http://blog.csdn.net/sinat_34200786/article/details/78090649

##How to use?

```
git clone 
```
```
cd Your_Path
```
```
scrapy crawl qi_dian_novel_spider
```
```
cd ../Wordcloud
```
```
python Wordcloud.py
```
